# Index

This directory contains files that are related to external documentation that might be used in
the verification/specification generation process.

- [ANSI-C Library Documentation](./ansi_c_library_documentation.json): This is a mapping from
    functions defined in the ANSI-C standard to their documentation.
    See [C Docs](https://github.com/jyoo980/c-docs) for information about its provenance.
